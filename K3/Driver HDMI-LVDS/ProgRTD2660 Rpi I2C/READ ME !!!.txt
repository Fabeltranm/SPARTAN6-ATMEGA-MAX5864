

 >>> Programar memoria flash de PCB800099 dede Raspberry Pi 


 >> Caracteristicas del firmware para el LCD LTN133AT16 302

  > Resolucion:	1366x768
  > Color Depth: 262K
  > Signal Type: LVDS (1CH, 6bit)  


 >> Pasos previos

  > Tener activo I2C de la raspberry
  > Ya contar con smbus para python: 

			$ sudo apt-get install -y python-smbus i2c-tools 

  > Al realizar el scanner de dispositivos I2C desde la raspberry:

			$ sudo i2cdetect -y 1
    
    Obtener las siguientes direcciones:

 

 >> Programar:

    En esta misma carpeta ejecutar desde consola:

			$ python main.py /ubicacion_del_firmware/firmware.bin

    Si va a utilizar el LCD LTN133AT16 302 utilizar el firmware: PCB800099-lvds136X768-d6bit-2AV1VGA1HDM1ACC-??1.5S.BIN
